CLDF v1.1

Added

New components - MediaTable - ContributionTable

with corresponding reference properties - mediaReference - contributionReference

and new properties - mediaType - downloadUrl - contributor - citation

Backward Incompatibilities

Expected cardinalities for some properties have been put in concrete terms in the Ontology and the component metadata. While this should be regarded largely as a bug fix, i.e. spelling out assumptions that were already implicit in the specification, this may lead to incompatibilities for datasets violating these assumptions.
